BankSys - Sistema de Gestión para Servicios Financieros

Descripción

Aplicación de escritorio desarrollada en Python que permite gestionar información de un sistema financiero.
El programa utiliza una interfaz gráfica con Tkinter y una base de datos MySQL para almacenar y administrar la información.

El sistema permite registrar y gestionar clientes, cuentas bancarias, créditos y transacciones de forma sencilla mediante formularios interactivos.


Tecnologías utilizadas
Python
Tkinter (Interfaz gráfica)
MySQL (Base de datos)
MySQL Connector Python**
Openpyxl (Exportación a Excel)
Visual Studio Code (Entorno de desarrollo)



Funcionalidades del sistema


Gestión de Clientes

Permite registrar información de los clientes como:

* Tipo de cliente
* Nombre o razón social
* Documento
* Teléfono
* Correo electrónico

Gestión de Cuentas

Permite administrar cuentas bancarias con información como:

* Número de cuenta
* Tipo de cuenta
* Moneda
* Sucursal
* Saldo
* Estado

Gestión de Créditos

Permite registrar créditos otorgados a clientes incluyendo:

* Número de operación
* Monto aprobado
* Plazo
* Tasa de interés
* Estado del crédito

Gestión de Transacciones

Permite registrar operaciones financieras como:

* Transferencias
* Movimientos entre cuentas
* Registro de montos
* Canal de la transacción
